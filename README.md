# take-home-project

# client
1. in terminal:


    npm install
2. create .env file


    REACT_APP_BASE_URL=http://localhost:3000
    REACT_APP_API_BASE_URL=http://127.0.0.1:8000
3. in terminal:


    npm start
# server
1. in terminal: 


    pip install -r requirements.txt
2. config sql database name, username, password in app/settings
3. in terminal:


    python3 manage.py runserver
